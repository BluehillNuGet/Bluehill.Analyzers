# BH0003: Prohibit reassignment of primary constructor parameters

The parameters of the primary constructor must not be reassigned.

## Code with violation

```csharp
public class TestClass(int i) {
    public void TestMethod() {
        i = 10;
    }
}
```

## Fixed Code

```csharp
public class TestClass(int i) {
    private int I = i;

    public void TestMethod() {
        I = 10;
    }
}
```
